# Customer Service

Customer Service is a Spring Boot application that manages customer information.
It provides REST APIs for creating, updating, retrieving, and deleting customer records.

## Features

Add new customers

Update existing customer details

Fetch customer information by ID or all customers

Delete customers

Uses Spring Data JPA with MySQL / H2 database

Integrated with ModelMapper for DTO conversions

Includes Swagger UI for interactive API documentation

Unit tests with JUnit 5 and Mockito
